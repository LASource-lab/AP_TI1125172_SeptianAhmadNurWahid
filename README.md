Deskripsi Singkat Program:
- Tanpa Prototipe: Mendefinisikan fungsi secara lengkap sebelum fungsi main(). Dengan cara ini, tidak perlu menulis baris prototipe secara terpisah.
- Parameter Default: memungkinkan kita memberikan nilai "cadangan" pada parameter. Jika saat dipanggil kita tidak memberikan nilai, maka fungsi akan menggunakan nilai default tersebut.
- Void Parameter: Jika sebuah fungsi tidak memerlukan input apa pun, bisa mengosongkan tanda kurungnya atau menuliskan kata kunci void.
- Lambda: fungsi "tanpa nama" (anonymous function) yang biasanya digunakan di dalam fungsi lain. Dalam pemrograman modern untuk tugas-tugas singkat.
-  Pass-by-Reference:  mendeklarasikan fungsi yang tidak hanya menerima nilai, tetapi bisa mengubah variabel asli yang dikirimkan menggunakan simbol ampersand (&).
